# FreeRooms
Because my current university often has it's reservable study-rooms fully booked, I took it upon myself to write a program to find all empty rooms, based on our calendar program - TimeEdit

You need to add a LINK.txt and a Rooms.json file. 
Link.txt should contain the 2 Links: The first one for all students/staff, and the second for all rooms.
Rooms.json should be based off of the Rooms-Example.json
